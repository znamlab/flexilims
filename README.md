# flexilims
Python wrappers around the flexilims API

# Flexilims description

The database software is written and maintained by the database team of the Crick. Flexilims web interface can be accessed at https://flexylims.thecrick.org/flexilims/. Mike Gavrielides wrote an API. His initial documentations can be found in the `docs` folder.

This repository is defining python wrappers to use the API.
